## üõ† Technical Implementation

### Single-Page Design

**Rationale**:
- ‚úÖ Fast load time (< 500ms)
- ‚úÖ No navigation confusion
- ‚úÖ Smooth scroll experience
- ‚úÖ Easy maintenance

**Trade-offs**:
- ‚ùå Long page (but organized with clear sections)
- ‚ùå All content loads at once (but only 30KB total)

### Inline CSS/JS

**Rationale**:
- ‚úÖ Zero external requests
- ‚úÖ No CORS issues
- ‚úÖ Guaranteed styling
- ‚úÖ Easy deployment

**Trade-offs**:
- ‚ùå No caching (but page is small)
- ‚ùå Harder to maintain (but stable content)

### Minimal JavaScript

**Features**:
1. Fade-in on scroll (IntersectionObserver)
2. Smooth scrolling for anchor links
3. That's it - no bloat!
